Swift编程入门（1）类型、常量和变量

2018/12/7 posted in  Swift入门

一直想做一个桌面应用，买了swift的书一直没有看，最近开始捡起来学习，顺便做做笔记，也许记录的不会特别详细，后面会在重复看的时候再来补充。

  • 类型
    • 字符串：String
    • 整型：Int
  • 常量与变量
    • 常量
    • 变量
    • 字符串插值

类型

字符串：String

import Cocoa
let towname:String  = "Tom"

整型：Int

常量与变量

对必须变化的实例用变量，其他都用常量。

常量

你经常需要创建值不变的实例。针对这种情况，可以用常量（constant）。顾名思义，常
量的值不能改变。
声明常量需要使用关键字let，

import Cocoa
let number:Int = 4
//number += 4 常量不能做值得改变

变量

声明变量需要使用关键字var，
变量的值可以改变

import Cocoa
let number:Int = 4
number += 4

字符串插值

字符串字面量中的\()语法表示占位符，可以访问一个实例的值并将其放入新字符串（或者
说“插值”）。比如，\(townName)访问常量townName的值并将其放入新的字符串实例。

import Cocoa

let numberOfStopLight: Int = 4
var population: Int = 5422
let townName:String = "Knowhere"
let townDescription = "\(townName) has a population of \(population) and \(numberOfStopLight) stoplights."
print(townDescription)
